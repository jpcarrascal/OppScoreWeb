# OppScoreWeb
Web app to compute and map Opportunity Scores.
Use it here: https://jpcarrascal.github.io/OppScoreWeb/

Uses:
- D3.js
- Save SVG as PNG: https://github.com/exupero/saveSvgAsPng
- CSV parser: https://github.com/d3/d3-dsv
